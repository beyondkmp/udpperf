# udpperf

## server

```
Usage of ./server:
  -host string
        host
  -port string
        port (default "37")
```

## client
```
Usage of ./client:
  -host string
        host (default "localhost")
  -num int
        numClients (default 5)
  -port string
        port (default "37")
  -second int
        secondSleep (default 1)
  -size int
        sizePacket (default 5)
```
